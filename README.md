#  Hangman(Шибениця)

**Лабораторна робота з курсу КОП** – інтерактивний додаток на React, який демонструє роботу з компонентами, маршрутизацію.

Цей проєкт є прикладом освітнього застосунку, де користувач може взаємодіяти з елементами інтерфейсу, переглядати сторінки.

---

## Опис гри / додатку

В грі можна:
- Переглядати списки даних у реальному часі
- Взаємодіяти з елементами UI (кнопки, картки)
- Вивчати структуру React-компонентів через **Storybook**

---

##  Структура проєкту

| Папка / файл          | Опис |
|-----------------------|------|
| `src/`               | Компоненти та логіка додатку |
| `src/components/`     | UI-компоненти React |
| `src/pages/`          | Сторінки додатку |
| `App.js`             | Головний компонент |
| `public/`             | Статичні файли |
| `docs/`               | Згенерована документація та відео |
| `LICENSE.md`          | Ліцензія проєкту |
| `PRIVACY_POLICY.md`   | Політика конфіденційності |
| `EULA.md`             | Ліцензійна угода користувача |
| `README.md`           | Цей файл |

##  Встановлення та запуск

| Крок | Команда |
|------|---------|
| 1. Клонувати репозиторій | `git clone https://github.com/Margaryts/KOP_Lab1.git` |
| 2. Перейти у директорію | `cd KOP_Lab1` |
| 3. Встановити залежності | `npm install` |
| 4. Запуск додатку | `npm start` |
| 5. Запуск Storybook | `npm run storybook` |
| 6. Генерація документації | `npx jsdoc -c jsdoc.json` |

---

##  Особливості

-  Адаптивні React-компоненти
-  Storybook з демонстрацією 2+ компонентів та 2–3 варіаціями для кожного
-  Cookie popup згідно з GDPR
-  Документація коду за допомогою **JSdoc**
-  Відео демонстрації локальної системи документації (`docs/video_demo.mp4`)
