Ви можете:
- Використовувати Програмне забезпечення для **навчальних і особистих цілей**

- Модифікувати код **тільки для власного навчального досвіду**

Ви **не маєте права**:
- Використовувати Програмне забезпечення для комерційних цілей без письмового дозволу
- Розповсюджувати або продавати Програмне забезпечення
- Модифікувати його для перепродажу
- Видаляти авторські позначки або повідомлення про права
- Реверс-інженерити або декомпілювати код (за винятком дозволеного законом)
- Використовувати Програмне забезпечення для створення конкурентних продуктів

Програмне забезпечення **ліцензується, а не продається**. Ліцензіар зберігає всі права, включаючи:
